Projet Petit Grand

Travail réalisé :

- Les nombreuses formes ont été réalisé avec leur couleur correspondante + des formes suplémentaires.

- Le jeu est fonctionnel

- Le gamePlay a été modifé avec des nouveautés :
	 - un nouveau concept prévu spécialement pour un seul joueur 15 cartes au lieu de 30
	 - un systeme de Joker visant à rendre le jeu plus simple ( essayez de l'utiliser en appuyant sur sa tête vous verez ^^ )
	 - le jeu est gagné lorsque le joueur a vidé sa pile
	 - si le paris est perdu les cartes sont remises sous la pile à découvrir ( il faut donc jouer de sa mémoire et des probabilités )

- L'interface a été repensée :
	- une jauge permettant de voir le nombre de cartes restantes
	- des symboles clair
	- le nouveau symbole Joker ( vous verez, il se modifie meme quand on l'utilise... )
	- un écran de victoire fort sympatique
	- l'icone représentant le charismatique Joker

- Petite feature bonus : on peut s'amuser à bouger les formes qui représentent les animaux présent dans le jeu ( appuyer et bouger )
